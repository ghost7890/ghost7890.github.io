1、使用net start MongoDB命令报错——服务没有响应控制功能

问题原因：MongoDB服务没启动，配置文件错误

右键查看属性

我们可以看到MongoDB默认的配置文件D:\MongoDB\bin\mongod.cfg，打开配置文件，注释掉最后一行的mp:，重启启动服务即可解决问题。
